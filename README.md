# Water-level-Indicator

A **Water Level Indicator** is a system that monitors and displays the level of water in a tank or container. It provides real-time information about whether the water level is **low, medium, or high**, helping users manage water usage efficiently.

### Working Principle

The system typically uses:

* **Probe sensors** or
* **Float switches**

to detect the water level at different points inside a tank. These sensors send signals to a control unit, which then indicates the current water level.


### How It Works

The Water Level Indicator operates on a simple mechanism:

1. Sensors are placed at different heights inside the tank
2. When water reaches a sensor, it completes an electrical circuit
3. The system detects this signal
4. The corresponding water level is displayed (Low / Medium / High)

### Key Features

* Real-time water level monitoring
* Simple and cost-effective design
* Prevents water overflow and dry running
* Easy installation and maintenance

### Applications

* Overhead water tanks
* Underground water storage
* Industrial water management systems
* Agricultural irrigation systems
